# The Winning Formula - Brand Guidelines

| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|

| **Papaya Orange** | `#FF8000` | 255, 128, 0 | Primary accent, CTAs, highlights |
| **Pure White** | `#FFFFFF` | 255, 255, 255 | Body text on dark, key elements |

## Typography

### Primary Font Stack
**Headings**: Inter (Bold, Semi-Bold)
- Clean, modern, highly readable
- Works well at all sizes
- Fallback: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

**Body**: Inter (Regular, Medium)
- Consistent with headings
- Excellent for data-heavy content
- Fallback: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

### Font Sizes
| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| H1 (Title) | 32px | Bold (700) | 1.2 |
| H2 (Section) | 24px | Semi-Bold (600) | 1.3 |
| H3 (Subsection) | 20px | Semi-Bold (600) | 1.4 |
| Body | 16px | Regular (400) | 1.6 |
| Caption | 14px | Regular (400) | 1.5 |
| Small | 12px | Medium (500) | 1.4 |

### Monospace (for data/code)
**Font**: JetBrains Mono or SF Mono
- Used for: numbers, code snippets, data points
- Fallback: 'Courier New', monospace

## Logo Usage

### Primary Logo
The "OW" data visualization mark:
- Use on dark backgrounds only
- Minimum size: 40px height
- Clear space: Equal to the height of the "O"

### Newsletter Header
Format: "THE WINNING FORMULA" wordmark
- All caps, Inter Bold
- Letter spacing: 0.1em
- With tagline below in regular weight

## Visual Style

### Overall Aesthetic
- **Dark mode first**: Dark navy backgrounds
- **Data-driven visuals**: Charts, graphs, numbers as design elements
- **Clean and minimal**: White space is important
- **Professional but approachable**: Not corporate, not casual

### Images & Graphics
- Use high-contrast images on dark backgrounds
- Charts should use brand colors (orange for highlights)
- Screenshots should have subtle drop shadows
- F1 content: Action shots, car details, data overlays

### Cards & Containers
- Background: `#2D4A6F` (Light Navy)
- Border radius: 8px
- Border: 1px solid `#64748B` (optional)
- Padding: 24px

### Buttons & CTAs
- Background: `#FF8000` (Papaya Orange)
- Text: `#FFFFFF` (White)
- Border radius: 6px
- Hover: `#FFB366` (Soft Orange)
- Padding: 12px 24px

## Email Template Structure

### Header Section
- Dark navy background (`#1E3A5F`)
- Logo/wordmark centered
- Pillar indicator (colored tag)

### Content Section
- Background: `#1E3A5F` or `#2D4A6F`
- Text: White (`#FFFFFF`)
- Links: Papaya Orange (`#FF8000`)
- Blockquotes: Left border in pillar accent color

### Data Visualization Section
- Dark background
- Orange/white chart elements
- Clear labels in white

### Footer Section
- Lighter background or subtle separation
- Social links
- Unsubscribe in smaller text
- "Data-driven thinking for F1, property, skincare, and life."

## Writing Style (Summary)

### Do
- Use "I" and "you" - personal tone
- Show your work (screenshots, numbers)
- Connect data to decisions
- End with actionable takeaways

### Don't
- Corporate jargon
- Walls of text without visuals
- Assume everyone knows basics
- Forget the human element

### Key Phrases
- "The winning formula for..."
- "The data says X, but here's the catch..."
- "Let's break this down..."
- "Here's what I'd actually do..."
